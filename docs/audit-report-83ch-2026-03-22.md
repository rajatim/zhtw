## 測試方法論

### 資料來源

1. **簡體中文**：hetushu.com（和圖書）— 使用 `scripts/scrape_novel.sh` 擷取
2. **繁體中文 Gold Standard**：hjwzw.com（黃金屋）— 獨立繁體版本（非 OpenCC 轉換）

### 比對演算法

1. 文字分段（以換行切割，過濾 ≤8 字元短行）
2. 段落模糊配對（比較前 30 字元，相似度 ≥ 50% 視為同一段落）
3. 逐字比對配對段落，統計一致/不一致字元
4. 差異分類：zhtw 更好 / gold 更好 / 中性差異

### 為什麼不是 100% 準確

剩餘 2.31% 差異主要來自：

| 差異類型 | 佔比 | 說明 |
|---------|------|------|
| 一對多歧義字 | ~60% | 衝/沖、複/復、幹/乾 等需上下文判斷 |
| 引號風格 | ~15% | 「」vs ""、『』vs '' |
| 來源文字差異 | ~15% | 兩版本用字/刪改不同 |
| 詞彙偏好 | ~10% | 資訊/信息、開啟/打開 等兩岸習慣差異 |

### 品質判定

- **zhtw 更好: 210 處** — 使用台灣標準（牆不用墻、鍊不用煉、係不用系）
- **gold 更好: 0 處** — 無漏轉或轉錯
- **結論**: zhtw 的台灣在地化轉換品質優於通用繁體轉換

### 發現並修復的問題

本次 83 章實測過程中發現並修復了 6 個問題：

| 問題 | 修復 |
|------|------|
| `周→週` 過度轉換（周知、周到 全轉錯） | 移除裸映射，保留 111 個複合詞 |
| `泡沫→泡棉` 語義錯誤 | 改為 `泡沫塑料→泡棉` |
| `才→纔` 古字（60 個 OpenCC 詞條） | 全部移除 |
| `合→閤` 古字映射 | 移至歧義排除清單 |
| `的士卒` Aho-Corasick 重疊 | 新增 3 字元保護詞 |
| `衆/爲` 異體字（38 個詞條） | 修正為標準 `眾/為` |
